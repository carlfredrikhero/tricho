# Tricho CSS framework

An CSS library with simplicity in focus.

Default vertical grid: 24px

 - [x] Headings
 - [x] Paragraph
 - [x] Container
 - [x] Box
 - [ ] Content
 - [ ] Buttons
 - [ ] Grid

## Content

Add `.content` to a block element and all child block elements will get the $gap (default 24px) bottom-margin
